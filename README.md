## English

### Overview

A mobile app written with Kotlin that helps you organize your filmography. Keep track of what you're watching, want to watch, and have watched.

### ‚ú® Features

**Easy navigation through three main lists:**
* **"Watching":** Currently watching movies and TV shows.
* **"Want to Watch":** Your watchlist.
* **"Watched":** Completed movies and TV shows with the ability to add ratings and notes.

**Flexible Settings:**
* Manage your collection.
* Add personal ratings and notes to each item.

**Notifications:**
* Get notified when new episodes of your favorite TV shows are released.

**Movie Search:**
* Built-in search functionality using an extensive database (TMDB).

### üöÄ How do I install it?

This is a code repository and might not look like other software that you're used to downloading and installing (if you're not familiar with programming). After that, run the installer. When the installer finishes, a shortcut to the application will appear on your desktop, when you click on it, the application should start normally.
